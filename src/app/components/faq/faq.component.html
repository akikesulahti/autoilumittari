Mikä on Autoilumittari?

Web-sovellus matka-ajan ja polttoaineen kulutuksen väliseen vertailuun. Sovellus on tehty osana Solidabis koodihaastetta 13.05.2021: https://koodihaaste.solidabis.com/

Miten se toimii? Tehtävänannon mukaisesti:

Autojen bensankulutus kasvaa 1.009 kulmakertoimella. Eli jos auton nopeus kasvaa 1km/h, niin bensankulutus kasvaa 1.009 kertaiseksi. Eri autojen bensakulutus 1km/h nopeudella on seuraava:

  • Auto A: 3l / 100km
  • Auto B: 3.5l / 100km
  • Auto C: 4l / 100km
 Kuka tämän on tehnyt?

Aki Kesulahti. Lähdekoodi: https://github.com/akikesulahti/autoilumittari

Tietosuojakäytäntö

Autoilumittari itsessään ei kerää mitään dataa käytöstä, käyttäjästä, verkkoliikenteestä eikä mistään muusta. Sivusto ei käytä keksejä eikä analytiikkatyökaluja.

Verkkosivu https://autoilumittari.web.app/ toimii Google Firebase Hosting ympäristössä, joka säilyttää käyttäjän IP osoitetta palvelun väärinkäytön estämiseksi: "Hosting uses IP addresses of incoming requests to detect abuse and provide customers with detailed analysis of usage data. Hosting retains IP data for a few months."
